## 图片格式建议

- LOGO和图标推荐使用SVG格式，确保在不同分辨率下清晰显示
- 照片和大型图像使用JPG格式，并进行适当压缩（质量80%左右）
- 需要透明背景的图像使用PNG格式

## 命名规则

- 使用小写字母
- 单词间用连字符"-"分隔
- 命名要简洁明了，反映图片内容

## 优化建议

- 所有图片应进行适当压缩，以提高网站加载速度
- 考虑为大图提供多种尺寸版本，以适应不同设备
- 可使用在线工具如TinyPNG进行无损压缩
